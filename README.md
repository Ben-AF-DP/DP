
# GitHub-fertige PDF-Website

So veröffentlichst du diese Seite mit GitHub Pages:

1. Neues **Public Repository** erstellen (z. B. `pdf-website`).
2. **Inhalt** dieses Ordners hochladen (nicht den Ordner selbst):
   - `index.html`, `styles.css`, `README.md`, Ordner `files/` mit `dienstplan.pdf` und `.keep`.
3. Repo → **Settings** → **Pages** → Source: Branch `main`, Folder `/ (root)` → **Save**.
4. Warte 1–2 Minuten. Deine URL: `https://DEINNAME.github.io/REPO-NAME/`.

## Eigene PDF einsetzen
- Ersetze `files/dienstplan.pdf` durch deinen echten Dienstplan (Dateinamen beibehalten → Links funktionieren automatisch).
- Für ein Archiv: Weitere PDFs in `files/` ablegen und in `index.html` verlinken.

Viel Erfolg!
